## Overview

Add automatic update capability to the Savant desktop application. When a new version is published to GitHub Releases, the app checks for updates on startup and prompts the user to install via Tauri's built-in updater.

---

## Current State

- **Tauri version:** 1.7 (via tauri crate in desktop Cargo.toml)
- **Current version:** 0.0.1
- **Build target:** `tauri build` produces MSI + NSIS bundles
- **Release location:** `target/release/bundle/msi/` and `target/release/bundle/nsis/`
- **Updater:** None — users must manually download and reinstall

## Implementation Plan

### Task 1: Add updater feature to Cargo.toml
**File:** `crates/desktop/src-tauri/Cargo.toml` (line 7)
```toml
tauri = { version = "1.7", features = ["api-all", "system-tray", "updater"] }
```

### Task 2: Generate Ed25519 keypair
**Command:** `tauri signer generate`
- Creates `~/.tauri/savant.key` (private key)
- Prints public key to stdout
- Public key goes in `tauri.conf.json` updater config
- Private key used to sign releases

### Task 3: Configure updater endpoint
**File:** `crates/desktop/src-tauri/tauri.conf.json`
Add `tauri.updater` section:
```json
"updater": {
  "active": true,
  "dialog": true,
  "endpoints": [
    "https://github.com/fame0528/Savant/releases/latest/download/latest.json"
  ],
  "pubkey": "<GENERATED_PUBLIC_KEY>",
  "windows": {
    "installMode": "passive"
  }
}
```

### Task 4: Add error handling for update check
**File:** `crates/desktop/src-tauri/src/main.rs` (in setup handler)
```rust
.setup(|app| {
    let handle = app.handle();
    tauri::async_runtime::spawn(async move {
        match handle.updater().check().await {
            Ok(update) => {
                if update.is_update_available() {
                    info!("Update available: {}", update.latest_version());
                    let _ = handle.emit_all("update-available", update.latest_version());
                }
            }
            Err(e) => {
                info!("Update check skipped: {}", e);
            }
        }
    });
    // ... rest of setup
})
```

### Task 5: Generate release manifest
**Command (manual or CI):**
```bash
# Sign the release
tauri signer sign target/release/bundle/nsis/Savant_0.0.1_x64-setup.exe -k ~/.tauri/savant.key

# Generate latest.json for Tauri updater
# Format:
{
  "version": "0.0.1",
  "notes": "Release notes here",
  "pub_date": "2026-03-22T20:00:00Z",
  "platforms": {
    "windows-x86_64": {
      "signature": "<GENERATED_SIGNATURE>",
      "url": "https://github.com/fame0528/Savant/releases/latest/download/Savant_0.0.1_x64-setup.exe"
    },
    "windows-x86_64-msi": {
      "signature": "<GENERATED_SIGNATURE>",
      "url": "https://github.com/fame0528/Savant/releases/latest/download/Savant_0.0.1_x64_en-US.msi"
    }
  }
}
```

### Task 6: Version bump before each release
**File:** `crates/desktop/src-tauri/tauri.conf.json` (line 10)
- Increment `package.version` before each release
- Updater compares this against `latest.json` version to detect updates
