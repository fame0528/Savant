# FID-20260319-CTX: 5-Layer CortexaDB Cognitive Architecture

## 1. Scope & Objective
Transition Savant's fragmented memory substrate (Fjall + ruvector-core) into a unified, AAA-grade Cognitive Engine powered by **CortexaDB**. Implement Contextual Memory Virtualization (CMV) via DAGs, and establish a Hive-Mind Arbiter with Cryptographic Privacy Boundaries and Shannon Entropy constraints (<1.5 bits).

## 2. Approach & Dependencies
- **Retain:** `fastembed` (for 384-dim semantic embeddings), `jsonwebtoken` (for privacy).
- **Remove:** `fjall`, `ruvector-core`.
- **Add:** `cortexadb` (Git dependency: `https://github.com/anaslimem/CortexaDB`).
- **Core Structural Shift:** `MemoryEngine` manages one private CortexaDB (Local Enclave) and one shared DB (Collective Graph).

## 3. Discovered Patterns
- **Auth:** Rely on JWT for boundary enforcement.
- **Async Execution:** `axum` routes operate highly concurrently; `tokio::sync::broadcast` handles global updates without blocking HTTP handlers.
