# FID-20260321-TUNE: AI Fine-Tuning Dashboard System

## 1. Scope & Objective

Implement a dedicated "Fine-Tuning" interface within the Savant Dashboard to allow real-time adjustment of global AI parameters (`top_p`, `frequency_penalty`, `presence_penalty`, etc.). This system will bridge the gap between technical configuration and artistic/conversational control, ensuring the "Stillness & Presence" directive can be tuned visually.

## 2. Approach & Dependencies

- **Infrastructure**: Extend `ConfigUpdateRequest` and `handle_config_set` in the Gateway to support the new AI parameters.
- **Frontend**: Create a new `Tune` page/module in the Next.js dashboard.
- **UI Components**: Implement sleek, glassmorphism-style sliders and numeric inputs for parameter control.
- **Real-time Sync**: Ensure changes persist to `savant.toml` and are immediately reflected in new agent sessions.
- **Verification**: `cargo check` (backend) + `npm run dev` (frontend) + manual tuning verification.
